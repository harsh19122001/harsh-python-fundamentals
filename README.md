# Python Fundamentals— Notebooks

## 01_python_basics
Collection of Jupyter notebooks covering Python fundamentals:
- data types, variables, and rules
- type casting and operators
- indexing, slicing and string operations
- imports, `math` module, and small exercises

## 02_conditional_statement
- Even–Odd number detection
- Positive / Negative / Zero classification
- Age category detection (Child / Teen / Adult / Senior)
- Pass / Fail system
- Grade evaluation (A, B, C, D)
- Temperature categorization (Cold / Mild / Warm / Hot)
- FizzBuzz logic
- Voting eligibility check (Age + ID verification)
- Range validation (1–10 inclusive)
- Login & password strength validation
- Nested conditions (e.g., Non-negative even/odd)
- Comparing two values (greater, smaller, equal)
- Boolean checks without logical operators

## How to run
1. Create virtual env: `python -m venv venv && source venv/bin/activate`
2. Install: `pip install -r requirements.txt`
3. Open notebooks: `jupyter lab` or `jupyter notebook`
